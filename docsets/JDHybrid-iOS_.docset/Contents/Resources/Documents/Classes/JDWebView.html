    或实现JDWebViewDelegate中的接口自定义JDWebView相关功能和逻辑 设置delegate时会设置delegateProxy为nil

  • customJsDelegate

    自定义的WebViewjs代理： 1、注意：WebView默认的会提供一个js代理，此代理用于处理公共的原生能力。如获取当前网络状态、获取手机基本信息等 2、但是，如果业务需要处理自定义的协议，比如导航栏、XView等，这些业务场景有一些自己特有的协议。则需要自定义js代理来处理 3、自定义的代理只处理自定义代理中实现了的方法，未实现的方法仍然由默认的代理实现。

    Declaration

    Objective-C

    @property (nonatomic, weak) id<JDWebViewJSBridgeProtocol> _Nullable customJsDelegate;

    Swift

    weak var customJsDelegate: (any JDWebViewJSBridgeProtocol)? { get set }
